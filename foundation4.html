Plain English Explanation

Information is conserved—recognition transforms but doesn't destroy data. Like quantum evolution, changes are reversible. Derives from positive cost: energy conservation implies info preservation.

Lean Code: Unitary Transform

structure UnitaryTransform (n : Nat) where
  forward : QuantumState n → QuantumState n
  backward : QuantumState n → QuantumState n
  left_inv : ∀ ψ, backward (forward ψ) = ψ
  right_inv : ∀ ψ, forward (backward ψ) = ψ

Explanation: Transformations that are reversible, preserving state.

Lean Theorem: Info Conservation

theorem information_conservation {A B : Type}
  (hA : Finite A) (hB : Finite B) (_ : Recognition A B) :
  ∃ (n : Nat), n = information_content A hA + information_content B hB := by
  refine ⟨hA.n + hB.n, ?_⟩
  rfl

Explanation: Recognition preserves total information amount.

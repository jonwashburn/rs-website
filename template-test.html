How This Template Works

This template extracts the clean, academic design from the constants page. It uses prefixed classes (template-*) so it won't break existing pages.

The spacing is intentionally tight (0.75rem sections) just like the constants page. The content width is locked at 900px for optimal readability. Typography follows a clear hierarchy with Space Grotesk for headings.

Template Components

All the reusable patterns are available: hero sections, content cards, buttons, and consistent spacing. Everything is built with CSS variables for easy customization.

This is the warm yellow/orange background variant. Perfect for highlighting special content or important callouts. Notice the subtle golden border.

How to Use This Template

To convert an existing page to this template, add the CSS file and update your HTML classes. The template- prefix ensures no conflicts.

Step 1: Add the CSS

<link rel="stylesheet" href="/assets/css/site-template.css">

Step 2: Add template-page class to body

<body class="template-page">

Step 3: Update your HTML classes

  • hero → template-hero
  • section → template-section
  • container → template-container
  • Add template- prefix to all component classes
